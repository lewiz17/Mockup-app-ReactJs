## Setup :pushpin:

```
git clone https://github.com/lewiz17/Mockup-app-ReactJs
cd Mockup-app-ReactJs
npm install
```

### Compiles and hot-reloads for development
```
npm run start
```

### Compiles and minifies for production
```
npm run build
```
